## Color

common roles and usage. This allows for uniform color application across themes
while maintaining full styling flexibility.

## Global

The other categories are global and component-specific variables. These control
more general styling of components, such as box shadow and border width.

```css dark
--border-width-base: 1px;
--border-style-base: solid;

--border-color-base: var(--color-app-mid);

--border-radius-xs: 2px;
--border-radius-sm: 4px;
--border-radius-md: 8px;
--border-radius-lg: 16px;
--border-radius-xlg: 50%;

--box-shadow-1: 0px 3px 9px -2px rgba(24, 58, 108, 0.1),
  0px 3px 4px -2px rgba(24, 58, 108, 0.1),
  0px 1px 1px -1px rgba(24, 58, 108, 0.14);

--box-shadow-2: 0px 5px 11px -5px rgba(24, 58, 108, 0.15),
  0px 6px 9px -5px rgba(24, 58, 108, 0.14),
  0px 1px 1px -1px rgba(24, 58, 108, 0.14);

--box-shadow-3: 0px 10px 24px -9px rgba(24, 58, 108, 0.118881),
  0px 12px 11px -13px rgba(24, 58, 108, 0.178322),
  0px 3px 17px -8px rgba(24, 58, 108, 0.190751);

--box-shadow-4: 0px 15px 38px -11px rgba(24, 58, 108, 0.15),
  0px 22px 7px -21px rgba(24, 58, 108, 0.14),
  0px 3px 17px -8px rgba(24, 58, 108, 0.433703);
```
