## Using Typography

- **Make it easy to scan** - To create visual groupings, keep related text aligned and closer together. The majority of text in most languages should be left-aligned.
- **Establish a visual hierarchy** - To create hierarchy, combine font weight, size, and color. To de-emphasize secondary content, use a lighter color or font size.

## Alignments

**Left aligned:** This is our default alignment. There are reasons to use left-aligned text by default:
- Because users read in "F" patterns, placing text in a similar pattern helps users absorb all of the information they need for a good experience.
- It is difficult for users to read center-aligned texts as it moves around; this can be especially difficult for people with dyslexia.

**Center aligned:** For very short blocks of content, such as text inside buttons or tabs, use center-aligned text.

**Right aligned:** Use right-align texts for numerical data in tables to make them easy to compare. This is the most common setting for right-to-left languages, such as Arabic and Hebrew.

## Typographic Hierarchy

The use of typographic hierarchy is one of the most significant approaches for successfully communicating material. Typographic hierarchy sets an order of priority that visually directs the reader through text to easily locate what they are searching for by using differences in size, weight, color, and location of type.
