## Border Radius Styles
For rounding common items such as cards, buttons, and messages, use our predefined border-radius value. Alternative values should be avoided except when creating custom shapes.

Please consult with SUI team when using alternative border-radius values aside from the following:

### border-radius-xs (2px)
Used for small elements that are 24px or less in height. Can also be used in specific cases, like when an element with small border-radius radius is "scaled down" in size.

<img src={extraSmall} alt="Extra small" width="846" />

### border-radius-sm (4px)
These are used for small UI elements, e.g. Buttons, Popover, Tooltips, Inputs, Dropdown, Table-cells, etc. Please use a border radius of **4px on anything 64px and under**.

<img src={small} alt="Small" width="846" />

### border-radius-md (8px)
These are used for medium to large UI elements, e.g. Accordion, Cards, Dialog, Toast, Media Gallery, Wizards, etc.

<img src={medium} alt="Medium" width="846" />

### border-radius-lg (16px)
Can be used for large elements and containers.

<img src={large} alt="Large" width="846" />

### border-radius-xlg (100%)
This style is used to makes the elements appear "pill-like".

<img src={extraLarge} alt="Extra large" width="846" />

## Using border-radius
- Be cautious when setting border-radius values in accordance with the rules outlined above.
- To prevent the shape from becoming warped, apply the same value to all corners.
- Combining border-radius values is not recommended.
