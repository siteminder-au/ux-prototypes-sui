## Accordion

Use only when there are no editable cells or other mandatory elements on the rate level.

Regarding collapsed/expanded: whilst there is no prescribed default, consider the following:
- Default to "collapsed" if use case should facilitate quick selection of all rates within a rate plan.
- Default to "expanded" if use case implies more selective "cherry-picking" on a rate level.

Accordion mode will always require checkboxes.
