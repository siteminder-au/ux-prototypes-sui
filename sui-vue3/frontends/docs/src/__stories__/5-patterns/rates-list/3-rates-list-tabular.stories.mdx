## Tabular

Use when there are editable cells or other mandatory elements on the rate level.

Note tabular view does not allow accordion functionality. This ensures the important/required fields are visible.

Checkboxes are optional in Tabular mode. They will need to be included when both selecting rates and editing rate level settings are required behaviours.
