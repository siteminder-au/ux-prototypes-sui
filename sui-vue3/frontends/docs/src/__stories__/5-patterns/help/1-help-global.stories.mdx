## App-header button

### Usage
The help icon in the global navigation directs users to the <strong>SiteMinder learning site</strong> which provides answers to common questions about <a href="https://learn.siteminder.com/s/" target="_blank"> SiteMinder</a>. Once selected it <strong>opens in a new tab</strong>.

### Formatting
#### Placement
Help content should be easy for users to find. To provide help for users, the help icon button is placed on the global navigation between the property switcher and notification icon button.

#### Anatomy
**1: Container** - The help icon button is typically displayed in a square container. It always has a fixed width and height and designed to hold a single icon.

**2: Icon** - Icons always have a fixed width and height. In this case we specifically use the icon-information icon.

### Specs
#### Help icon button
Help icon button is 48 x 48px.

## Help text link

### Usage
The help text link directs users to a specific page in the SiteMinder learning site which provides answers to common questions related to the content on the page. Once selected it opens in a new tab.

Please use the text button variant of the button component when placing these contextual links in your designs.

### Formatting
#### Anatomy
This link uses the text button variant of the button component.

**1: Prefix icon** - Prefix icon appears before the text. For contextual help links use  information icon as prefix. For help guides use information-alt icon as prefix.

**2: Text** - For contextual help, use this copy:
- “Help with [page name]”
- For help guides use this copy:  “Help guide”

<img src={globalTextLinkAnatomy} className="img-padding" alt="Anatomy of help text link" width= "240"/>

#### Placement
For the help link we use a text button with a prefix icon. <strong>Contextual help links are placed in In-App pages</strong> on the header section opposite side of the page name in order to relate to the content in which they appear.

#### Specs
##### Help icon button
The size of the text button is 40px high.
