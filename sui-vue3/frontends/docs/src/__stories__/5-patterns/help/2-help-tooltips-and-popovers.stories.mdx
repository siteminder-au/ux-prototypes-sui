## Tooltips
### When to use
Tooltips are great for giving more contextual information to a user. For instance a new button might be introduced with an uncommon icon in it, pairing it with a tooltip on hover to further explain what the button does would be a great way to boost the user experience.

## Popovers
### When to use
Use popovers when you want to convey rich and/or interactive content to the user in a non-disruptive format.
<img src={popoversWhenToUse1} className="img-padding" alt="A popover is used as help in side navigation menu" width="720"/>

When using a popover with a trigger element that is an icon (see below) please ensure that the icon used is the `utility-information-alt` icon in the colour token `$grey-neu-dark` (#717171).
