## Formatting

### Anatomy

**1: Image** - This SVG should have a max-width set of 1366px, to keep the height from increasing too much on larger screens.

**2: Header** - The header used here should be H3.

**3: Copy** - Please use default body styling for this: para-lg.

**4: Actions** (optional) - Actions can consist of a primary or secondary button. This is optional, however to reduce user frustration, please try to offer an action to assist in resolving the issue.

<img src={formattingImage} alt="Anatomy" />

### Sizing

Full page error pages should not display anything but the error design. The image size should hit a max-width of 1366px.
