## Usage
  ### 1. To separate or divide dense content
  They help improve readability and scannability by creating visual breaks between content.<br />
  They also indicate a shift in subject, content style, or anticipated action.
  <img src={usageDivideContent} className="img-padding" alt="Divider separating dense content" width="620" />

  ### 2. To create space between elements in a layout.
  Used to separate different sections or elements on a page or screen, such as between content, forms, or navigation items in a menu. <br />
  They are helpful for highlighting sections or lists, especially on smaller screens.
  <img src={usageCreateSpace} className="img-padding" alt="Divider creates space between layouts" width="560" />

  ### 3. Creating distinctions within related data.
  Dividers can also be used to create a sense of hierarchy, indicating relationships between elements (e.g., grouping related items).
  <img src={usageCreateDistinction} className="img-padding" alt="Divider creates distinction within related data" width="720" />

  ### Spacing
  Ensure that there is consistent spacing around the divider. It should be evenly spaced from adjacent content or elements to create a balanced visual effect.

  A **24-32px spacing** is recommended between elements and dividers.

  The **sm-divider** by default already has a **16px margin-top and bottom**, so consider these for the recommended spacing given above.

  ### Alignment

## Best practices
- Ensure dividers are noticeable but not too prominent.
- It's best to use them in moderation within a layout to prevent unnecessary clutter and buildup. Only resort to using dividers when open space can't group items together.
- Use dividers for grouping elements, not for separating individual items.

## Accessibility
Dividers are decorative elements and have no minimum contrast requirements.

  ### Reference
  [ARIA: separator role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/separator_role) <br />
  The separator role indicates the element is a divider that separates and distinguishes sections of content or groups of menu items.

  ### Screenreader expectations:
  For screen reader users, the experience should match what they see visually, indicating the start of a new "section" when a divider is shown.

  ### Keep in mind
  The divider doesn't notify a screen reader; it only briefly pauses the content being read aloud.
