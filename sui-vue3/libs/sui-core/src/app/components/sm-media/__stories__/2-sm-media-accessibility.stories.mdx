## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Tab | First focusable element within the component. |
| Shift + Tab | Move focus to the previous element. |
| Enter | Triggers the currently focused element. <br /><br />If focused on the image, this will select the image. <br /><br />If focused on the action button, this will trigger the action. |
