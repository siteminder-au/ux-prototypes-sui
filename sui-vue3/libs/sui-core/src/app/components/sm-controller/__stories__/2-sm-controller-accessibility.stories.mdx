## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Tab | Moves focus to next item. |
| Space<br />Enter | Activates the custom area item and moves to focus on the first item in the triggered area. |
| Escape | Closes the menu. <br />Sets focus to the controller button. |
| Left Arrow | Moves focus to the previous menu item. |
| Right Arrow | Moves focus to the next menu item. |
| Home | Moves focus to the first menu item. |
| End | Moves focus to the last menu item. |


## Focus Order
Make sure that the focus flows logically from one element to the next.
