## Usage
### When to use
- When switching between different columns of a table (e.g. calendar, reservations)
- Table control is recommended for tables with overflow. It is generally not required on tables with columns that fit within the page container.

## Formatting
### Anatomy
**1: Button icons - **Provides access to other columns while indicating that more columns exist within the table.Controls display left and right arrows for navigating the table.

Example of setting for button icons:
- left-alt icon shows previous week/#weeks
- left icon shows previous day
- right-alt icon shows next week/#weeks
- right icon shows next day

**2: Custom area -** The custom area is a text button that can be used to add a date picker or show/hide columns.

**3: Container -** Wraps the icons and custom area.

By default, icons and custom area are rectangular. The container has rounded corners with a 4px corner radius.

### Elevation
**Elevation settings:** 0 1px 1px 0  rgb(0 57 85 / 19%)
- Color: #003955
- X:0
- Y: 1
- Blur: 1
- Spread: 0
