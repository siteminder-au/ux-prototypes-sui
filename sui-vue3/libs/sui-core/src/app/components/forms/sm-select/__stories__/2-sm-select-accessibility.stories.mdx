## Keyboard navigation

### Select
| Key | Usage |
| :--- | :--- |
| Tab | Activates the select menu and moves to focus on the first item in the dropdown list.<br /> <br />Closes menu and takes user to the next item in tab order. |
| Escape | Closes the dropdown list when it is in an open state and the currently selected item remains unchanged. |
| Enter | Sets the textbox value to the content of the selected option.<br /> <br />Deselects the textbox value to the content of the selected option. <br /><br /> Activates menu item, causing action to be executed. |
| Escape | Closes the dropdown list when it is in an open state and the currently selected item remains unchanged. |
| Down Arrow | Set focus on the first item in the multi-select when no item is selected. Otherwise, moves focus next to the currently selected item. |
| Up Arrow | Moves focus to previous item. |
| Home | Moves focus to first option. <br /><br /> Optionally, in a single-select listbox, selection may also move with focus. Supporting this key is strongly recommended for lists with more than five options. |
| End | Moves focus to last option. <br /><br /> Optionally, in a single-select listbox, selection may also move with focus. Supporting this key is strongly recommended for lists with more than five options. |
| Type-ahead: is recommended, especially those with more than seven options | Type a character: focus moves to the next item with a name that starts with the typed character. <br /><br /> Type multiple characters in rapid succession: focus moves to the next item with a name that starts with the string of characters typed. |
| Shift + Tab | This key combination provides the same behavior as the Tab key, but in the reverse order. |
| Printable characters | If `filterable` is on then the list will filter by character |

### Tag
| Key | Usage |
| :--- | :--- |
| Tab | Takes user to the next item in tab order. |
| Enter | When focused on the tag, removes the filter. <br /><br /> Focus should then be placed on the nearest option:<br /><br /> <li>When on the last tag, place focus on the previous tag</li><li>When on the first tag, place focus on the next tag </li><li>When on the only tag, place focus on the listbox or any input element the tag might be associated with.</li> |
| Home | Moves focus to first item. |
| End | Moves focus to last item. |
| Shift + Tab | This key combination provides the same behavior as the Tab key, but in the reverse order. |
