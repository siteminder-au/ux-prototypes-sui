## Usage
Use the calendar component when you want to allow users to open a calendar popover for selecting a specific month and year. This helps the user visually and conveniently choose the desired month and year in context with other possible options.

<div className="sui-storybook-highlight sui-storybook-highlight--alert">
SUI uses two different components for date picking: <strong>sm-date-picker</strong> and <strong>sm-calendar</strong>. <br /><br />
<strong>sm-date-picker</strong> uses a plugin that is very feature rich. Currently this is the main component to use for selecting date and time. <br /><br />
<strong>sm-calendar</strong> has been built by the SUI team so we own the code. Currently this is only used for month and year picking, however we want to eventually build all our features into this component and depreciate <strong>sm-date-picker</strong>.
</div>


## Formatting

### Anatomy
**1: Input field (trigger)** - The input field works similarly to our sm-select field. This will trigger the dropdown. The icon placement in the select field inherits the same rules as the icon in our input field component: prefix, suffix, or both.

**2: Header** - The header contains button pagination as well as a sub-picker that contains a year picker.

**3: Year Selector** - Enables user to select the desired year directly.

**4: Current month / year** - The users current month / year.

**5: Selected month / year** - The selected month / year.

**6: Calendar grid** - This is the grid that allows a user to keyboard tab using up, down, left and right arrows.

<img src={anatomy} className="img-padding" alt="Calendar anatomy" width="720px"/>

### Placement
By default, the calendar dropdown will appear below the select field.

### Sizing
The width of the select input is adjustable by your engineers.

The width of the calendar dropdown is currently not adjustable. It will be width 100% of the **parent container** the user drops it in.

### Month and year: Masks
The displayed text on the input can be masked via masks prop.
For instance: :masks="{ input: 'MMM YYYY' }"

View available tokens at  [sm-calendar: Month and year: Masks](https://sui-dev-v3-sui-docs.dev.siteminderlabs.com/?path=/docs/components-form-select--standard)

## Behaviour
### Motion
Upon triggering the calendar dropdown via the select field, the dropdown will animate in using the standard sui ease of 0.3ms.
