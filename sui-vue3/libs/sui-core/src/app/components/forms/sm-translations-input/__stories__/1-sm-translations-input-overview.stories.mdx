## Usage
Users can enter text in one language, and be able to add its translations  into their desired language, providing a convenient way to translate content within an application.
<div className="sui-storybook-highlight sui-storybook-highlight--warning">This component is only used by Multi-property</div>

## Formatting

### Anatomy
**1: Label** - Labels should show what information is needed in the field. The label is aligned to the left.

**2: Text area input** - A form input field designed for entering text to be translated

**3: Connector** -  Shows connections between the text area input and other translation inputs.

**4: Translation label** - This contains all the available times for selection. By default time intervals of 1 hour is displayed. Customisation is possible through the step props.

**5: Translation input** - A form input field designed for entering translated text.

**6: Delete button** - Enable delation of added translation.

**7: Add translation button** - Enables users to add more language type translation.

**8: Dropdown** - Enables users to select supported translations.
