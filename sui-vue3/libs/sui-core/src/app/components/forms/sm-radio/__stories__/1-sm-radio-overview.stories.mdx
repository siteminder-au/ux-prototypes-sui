## Usage
Use radio buttons when you have a collection of mutually exclusive options and users are allowed to pick one from the group.

### When to use
  - When you want the user to view all possible alternatives, use radio buttons. Consider using a dropdown or select component if available selections can be compacted to save space.
  - When users must evaluate and compare choices before making a decision.
  - Use radio controls if you have less than 7 options.

### Where to use
  - **Forms -** Can be utilized in full-page forms, modals, or side panels.
  - **Settings -** In a menu, page, or component, this is used to switch from one setting to another.
  - **Choosing from a list -** To choose a single item from a list.

<div className="sui-storybook-highlight">Consider using a select component instead of a radio button if available options can be compressed to save space.</div>

### Placement
The radio button is frequently used in forms. It may be added to a full page, a modal, card, dialog or a side panel. A radio button may also be used to change menu settings or to pick a single item in a data table.

For more information on spacing in forms, etc. , see our [spacing guidance](https://sui-dev-v3-sui-docs.dev.siteminderlabs.com/?path=/story/foundations-spacing-overview--page).
<img src={spacing} className="img-padding" alt="Placement for radio button" />


### Orientation
Radio buttons can be arranged in horizontal or vertical orientations so that users may choose from a range of related items.
<img src={orientation} className="img-padding" alt="Vertical and horizontal orientations for radio button" />


### Overflow content
- Allow text to wrap beneath the radio button to keep the control and label top aligned.
- Do not vertically center align text with the radio button.
- Consider rewording the label if space is limited.

## Types

### Standard
If only one item from a list may be selected, radio buttons should be used instead of checkboxes.
<img src={typeStandard} className="img-padding" alt="Standard radio button sample in rate setup" />

### Radio button group
- Controls are used to instantly choose between a limited number of mutually exclusive items.
- Only one option may be selected and activated at the same time.
- The results should be shown quickly once a user picks an option.
- Provide a clear label that describes the function of the radio button group.
