## Keyboard navigation


### Radio button
[https://www.w3.org/WAI/ARIA/apg/patterns/radiobutton/](https://www.w3.org/WAI/ARIA/apg/patterns/radiobutton/)

| Key | Usage |
| :--- | :--- |
| Tab | Moves keyboard focus to the checked radio button in a radio group.<br /><br />If no radio button is checked, focus moves to the first radio button in the group |
| Down arrow | Moves focus to next radio button in the group.<br /><br />If focus is on the last radio button in the group, move focus to the first radio button. |
| Up arrow | Moves focus to previous radio button in the group.<br /><br />If focus is on the first radio button in the group, move focus to the last radio button. |


### Radio button group
[https://www.w3.org/WAI/ARIA/apg/example-index/toolbar/toolbar.html](https://www.w3.org/WAI/ARIA/apg/example-index/toolbar/toolbar.html)

| Key | Usage |
| :--- | :--- |
| Tab | Moves keyboard focus among the radio button group. |
| Shift + Tab | Goes to the previous element. |
| Right Arrow	 | Moves focus to the next control. <br /><br />If the last control has focus, focus moves to the first control. |
| Left Arrow | Moves focus to the previous control. <br /><br />If the first control has focus, focus moves to the last control. |
| Enter <br />Space | Activate the button in focus |
