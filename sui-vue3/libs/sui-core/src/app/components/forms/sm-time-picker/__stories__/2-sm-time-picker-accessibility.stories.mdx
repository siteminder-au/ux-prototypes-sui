## Keyboard interaction
| Key | Usage |
| :--- | :--- |
| Tab | Moves focus to next focusable element. <br /><br />Automatically opens the time picker dropdown. |
| Shift + Tab | Moves focus to previous focusable element.|
| Escape | 	Closes the picker |
| Enter | Selects the highlighted time in the dropdown. |
| Up arrow (in dropdown) | Moves focus to the time above. |
| Down arrow (in dropdown) | Moves focus to the time below. |
