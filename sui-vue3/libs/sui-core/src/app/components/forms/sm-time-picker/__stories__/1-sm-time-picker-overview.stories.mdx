## Usage
The time picker lets users to either choose a single time from a menu or input a specific time as needed for scheduling. It's frequently used in forms that enables users to specify a time.
### Scheduling
Time pickers are often used for scheduling, whether users need to choose a particular time or establish a range of availability.
<img src={usage} className="img-padding" alt="Time picker used for check-in and check-out." width="640px"/>

## Formatting
### Anatomy
**1: Label** - Labels should show what information is needed in the field. The label is aligned to the left.

**2: Time input field** - Choosing this field reveals a dropdown where available times are displayed. Upon selecting a time, it will appear in the text input. The default time is a 24-hour format, customization is possible using the `from` and `to` props.

**3: Time selection dropdown** - This contains all the available times for selection. By default time intervals of 1 hour is displayed. Customisation is possible through the `step` props.

<img src={anatomy} className="img-padding" alt="Time picker anatomy." width="640px"/>

### Placement
By default, the time selection dropdown will appear below the select field. If there is no space below the field in the viewport, then it will appear above the field.

### Sizing
The width of the select input is always 100% and can be controlled via width of a parent container.
