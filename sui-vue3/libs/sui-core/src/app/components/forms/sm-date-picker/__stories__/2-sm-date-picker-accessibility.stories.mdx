## Keyboard interaction
| Key | Usage |
| :--- | :--- |
| Tab | Moves focus to next focusable element inside the picker. <br /><br />When focus is on the last focusable element in the dialog, moves focus to the first focusable element in the picker. |
| Shift + Tab | Moves focus to previous focusable element inside the picker. <br /><br />When focus is on the first focusable element in the picker, moves focus to the last focusable element in the picker.|
| Escape | Closes the picker. |
| Enter | Triggers the selected element. |
| Up arrow (in calendar grid) | Moves focus to the date above. |
| Down arrow (in calendar grid) | Moves focus to the date below. |
| Left arrow (in calendar grid) | Moves focus to the date to the left. |
| Right arrow (in calendar grid) | Moves focus to the date to the right. |
