## Keyboard navigation


### Checkbox (simple)
[https://www.w3.org/WAI/ARIA/apg/example-index/checkbox/checkbox.html](https://www.w3.org/WAI/ARIA/apg/example-index/checkbox/checkbox.html)

| Key | Usage |
| :--- | :--- |
| Tab | Moves keyboard focus to the checkbox. |
| Space | Toggles checkbox between checked and unchecked states. |


### Checkbox (mixed state)
[https://www.w3.org/WAI/ARIA/apg/example-index/checkbox/checkbox-mixed.html](https://www.w3.org/WAI/ARIA/apg/example-index/checkbox/checkbox-mixed.html)

| Key | Usage |
| :--- | :--- |
| Tab | Moves keyboard focus among the checkboxes. |
| Enter <br /> Space | Cycles the tri-state checkbox among unchecked, mixed, and checked states. <br /><br /> When the tri-state checkbox is unchecked, all the controlled checkboxes are unchecked. <br /><br />When the tri-state checkbox is mixed, the controlled checkboxes return to the last combination of states they had when the tri-state checkbox was last mixed, or to the default combination of states they had when the page loaded. <br /><br />When the tri-state checkbox is checked, all the controlled checkboxes are checked. |


### Checkbox button group
| Key | Usage |
| :--- | :--- |
| Tab | Moves keyboard focus among the checkbox button group. |
| Shift + Tab | Goes to the previous element |
| Enter <br /> Space | Activates the element in focus |
| **Screen reader (expected action)** | Reads: "Button name (or aria-label), button" |
