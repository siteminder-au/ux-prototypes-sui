## Usage

A checkbox is an input control that enables a user to choose one or more items from a list. Checkboxes can also turn an option on or off.

### When to use

  - If the user may select multiple options, use checkboxes. Use radio buttons if the user can only choose one option.
  - Display a list with sub-selections.
  - In a desktop environment, toggle an item on or off
  - When used for ON/OFF or ENABLED/DISABLED situations that happen, require review or saving. An action such as SAVE, APPLY, NEXT, etc, is required before displaying results.
  - When you have an indeterminate state where you may pick all, none, or some actions, use checkboxes.

<div className="sui-storybook-highlight">If you are designing for mobile, do not use a single checkbox in any scenario for ON/OFF situations.</div>

### Where to use

  - **Forms -** Can be utilized in full-page forms, modals, or side panels.
  - **Filters -** Filters data on a page, in a menu, or within a component.
  - **Tables -** Checkboxes may be placed in the data table for bulk editing, selecting, etc.
  - **T&Cs -** Users can indicate whether you agree to the conditions by checking or unchecking the checkbox entry.
  - **Lists and sub-selections -** Checkboxes can have parent-child relationships with one another. When the main checkbox is selected, all of the sub-checkboxes are selected. When a main checkbox is cleared, all sub-checkboxes are cleared. If some, but not all of the sub-checkboxes are selected, the main checkbox becomes indeterminate/mixed.

  ### Grouped checkboxes
  Checkboxes can have parent-child relationships with one another.
  <img src={typeGroupedCheckbox} className="img-padding" alt="Checkboxes with parent-child relationships"/>

  ### Checkbox button group
  Checkbox button group displays several selected options at the same time. They combine sets of buttons into toolbars or split buttons. These are an alternative to checkboxes. The full benefit of the component is shown when all choices are displayed in the viewport at the same time.
  #### When to use:
  Use it when there are 4 - 7 selections available. For larger amounts of options,  use other components like checkbox, multi-select, or dropdown.
