## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Tab | When focus moves into the tab list, places focus on the active tab element. <br /><br />When the tab list contains the focus, moves focus to the next element in the tab sequence. |
| Enter <br />Space | When a tab has focus, activates the tab, causing its associated panel to be displayed. |
| Right arrow | When a tab has focus: <li>Moves focus to the next tab.</li><li>If focus is on the last tab, moves focus to the first tab.</li> |
| Left arrow  | When a tab has focus: <li>Moves focus to the previous tab.</li><li>If focus is on the first tab, moves focus to the last tab.</li> |
| Home | When a tab has focus, moves focus to the first tab. |
| End | When a tab has focus, moves focus to the last tab. |
