## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Tab | Move forward to the next card |
| Arrow keys | Move the focus within the card to focusable elements. <br /><br />If a button is provided on the cards, it should be given priority. If there are numerous buttons, the primary button should be the first one to receive focus. |
| Shift + Tab | Move focus backward to the previous card |
| Enter | If the card is clickable, pressing enter will open the card.<br/><br />If the card is not clickable but with buttons, and the buttons or another component in the card is focused, that action is performed. |
| Space | Space bar will change selection if the card is selectable. |
