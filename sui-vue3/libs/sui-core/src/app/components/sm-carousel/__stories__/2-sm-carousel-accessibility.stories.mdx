## Keyboard navigation
| Key | Usage |
| :--- | :--- |
| Tab | Move to the next focusable element (previous, next or dot counters). |
| Shift + Tab | Move to the previous focusable element. |
| Enter <br /> Space| Allows user to move through the collection of images by moving forward or backward or jumping to an image. |
