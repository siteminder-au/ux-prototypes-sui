## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Space | Opens drawer and moves to focus on close icon button. |
| Enter | Opens drawer and moves to focus on close icon button. |
| Escape | Closes drawer and moves focus to trigger action. |
| Tab | Moves focus to the next item in the drawer. |
| Shift + Tab | This key combination provides the same behaviour as the Tab key, but in the reverse order. |

## Tab order

1. Close button
2. Content area, in order
3. Save button
