## Keyboard interaction
| Key | Usage |
| :--- | :--- |
| Tab | Moves focus to next focusable element inside the dialog. <br /><br />When focus is on the last focusable element in the dialog, moves focus to the first focusable element in the dialog. |
| Shift + Tab | Moves focus to previous focusable element inside the dialog. <br /><br />When focus is on the first focusable element in the dialog, moves focus to the last focusable element in the dialog.|
| Escape | Closes the dialog. |
| Enter | Triggers the selected element. |
