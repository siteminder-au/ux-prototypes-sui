## Scaling and adaptation
When a fullscreen dialog sits within a wizard component, it can expand to a **maximum width of 1301px** and a **maximum height of 902px**. These maximum dimensions have been chosen so that the modal sits in the center of the wizard component with a **32px margin** around it on desktop. When a dialog content exceeds its maximum width and height, it scrolls vertically to accommodate content.
<img src={scalingFullScreenDesktop} className="img-padding"alt="Scaling and adaptation of custom dialog on desktop"/>

Dialogs should be kept **at least 24px** away from the screen's leading and trailing edges. On tiny displays, dialogs scale vertically.
