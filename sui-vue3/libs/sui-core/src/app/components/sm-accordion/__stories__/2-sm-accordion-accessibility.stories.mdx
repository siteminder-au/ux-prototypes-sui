      <td><p>Down Arrow (Optional)</p></td>

    </tr>
    <tr>
      <td><p>Up Arrow (Optional)</p></td>
      <td><p>If focus is on an accordion header, moves focus to the previous accordion header. If focus is on the first accordion header, either does nothing or moves focus to the last accordion header.</p></td>
    </tr>
    <tr>
      <td><p>Shift + Tab</p></td>
      <td>
        <p>Moves focus to the previous focusable element.</p>
        <p>All focusable elements in the accordion are included in the page Tab sequence.</p>
      </td>
    </tr>
    <tr>
      <td><p>Home (Optional)</p></td>
      <td><p>When focus is on an accordion header, moves focus to the first accordion header.</p></td>
    </tr>
    <tr>
      <td><p>End (Optional)</p></td>
      <td><p>When focus is on an accordion header, moves focus to the last accordion header.</p></td>
