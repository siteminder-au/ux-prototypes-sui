## Keyboard interaction

| Do | Don't |
| :--- | :--- |
| Escape | Dismisses the Tooltip |
| ***Note*** | Focus stays on the triggering element while the tooltip is displayed. <br /><br />If the tooltip is invoked when the trigger element receives focus, then it is dismissed when it no longer has focus. If the tooltip is invoked with mouseIn, then it is dismissed with on mouseOut. |
