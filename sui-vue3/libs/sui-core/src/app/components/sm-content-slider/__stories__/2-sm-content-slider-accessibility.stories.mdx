## Keyboard interaction

| Key | Usage |
| :--- | :--- |
| Tab| Move to the next focusable element (previous button, content or next button) |
| Shift + Tab | Move to the previous focusable element |
| Enter<br />Space | Allows user to select the buttons to move through a collection of images by moving forward or backward. |
| Right Arrow  | Goes to the next slide |
| Left Arrow | Goes to the previous slide |
